DARWIN :: Data Asset Resource Web Interface

Data Asset Resource Web Interface is an open-source, multi-level, architectural project tracking system designed especially for enterprise project management operating environments. Its primary goal is for keeping track of project names, acronyms, and descriptions, support contact personnel, software and hardware requirements, application middleware and external vendor information for different environmental aspects.

Applications Utilized: HTML, PHP, MySQL.
